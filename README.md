# Nuestro Primer Año

Experiencia web de aniversario para Fabian y Daira, hecha con HTML, CSS y JavaScript puro.

## Personalizarla

1. Crea la carpeta `assets/images/` y agrega fotografías optimizadas en WebP.
2. Edita `data.js` para cambiar capítulos, fechas, frases y rutas de fotografías.
3. Para la portada y el final, agrega en `index.html` a `.cover` y `.final-photo`: `style="background-image:url('assets/images/foto.webp')"`.
4. Reemplaza el texto de la carta directamente en `index.html`.
5. Crea `assets/audio/` y agrega `cancion1.mp3` y `cancion2.mp3`. El reproductor nunca usa autoplay.

Abre `index.html` para probarlo. GitHub Pages puede publicarlo sin compilación ni dependencias.
